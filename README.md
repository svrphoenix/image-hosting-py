# Проект "Сервер зображень"

Сервіс для перегляду та зберігання зображень у вигляді веб-додатку. Суть проста:
користувач заходить на сайт, завантажує фотки чи картинки, а натомість отримує
прямі посилання на них. Ці посилання можна використовувати де завгодно — у
соцмережах, блогах або просто надіслати друзям.

Користувачі можуть:

- Завантажувати фотки в популярних форматах (.jpg, .png, .gif).
- Бути впевненими, що їхні зображення надійно збережені.
- Отримувати зручні прямі посилання на завантажені зображення.
